
📊 Dashboard de Vendas – Excel

Dashboard interativo de análise comercial desenvolvido no Microsoft Excel, com foco em visualização estratégica de dados e tomada de decisão.

O projeto inclui:

📈 Indicadores de desempenho (Faturamento, Lucro, Ticket Médio)

📊 Análise por produto, vendedor e região

📅 Filtros dinâmicos por período

📌 Tabelas dinâmicas e segmentações de dados

🎯 Visual profissional voltado para gestão comercial

Este projeto foi desenvolvido com foco em portfólio para demonstrar habilidades em:

Tratamento e organização de dados

Modelagem com tabelas dinâmicas

Criação de KPIs estratégicos

Construção de dashboards executivos
